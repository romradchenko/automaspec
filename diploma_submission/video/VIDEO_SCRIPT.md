## Recording Tips

### Screen Recording
- Use OBS Studio or similar (1080p, 60fps)
- Set browser zoom to 100%
- Use a clean browser profile (no extensions visible)
- Pre-login before recording

### Audio
- Record narration separately if possible
- Use a quiet room
- Speak clearly and at moderate pace

### Editing
- Add smooth transitions between sections
- Include text overlays for key points
- Add background music (optional, low volume)
- Export as H.264 MP4

### Timing
- Practice the flow before recording
- Each section should flow naturally
- Don't rush — 2 minutes is plenty of time
